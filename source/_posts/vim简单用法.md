## vim 简单用法

> 默认 vim 打开后是不能录入的，需要按键才能操作，具体如下：

#### 1. 开启编辑：

```
按“i”或者“Insert”键
```

#### 2. 退出编辑：

```
按Esc按键
```

#### 3. 退出 vim：

```
:q
```

#### 4. 保存 vim：

```
:w
```

#### 5. 保存退出 vim：

```
:wq
```

#### 6. 不保存退出 vim：

```
:q!
```
